>[success] # 出现频率最高的元素 -- 总结
~~~
1.使用对象记录元素出现频率在依次比较得到出现频率最高的元素
~~~
>[danger] ##### 案例
~~~
1.思路先将数据结构变成对象，统计出某个值的出现频率'{ a: 3, b: 1, c: 1 }'
2."[ [ 'a', 3 ], [ 'b', 1 ], [ 'c', 1 ] ]"在当前值和之前值做比较看出现频率大小，采用reduce
~~~
~~~
const {log} =console

const mostFrequest = (arr)=>
    Object.entries(
        arr.reduce((acc,cur)=>{
        acc[cur] = acc[cur]? acc[cur]+1 : 1 
        return acc
        },{})
    ).reduce((a,v)=>(v[1]>=a[1]?v:a),[null,0])

log(mostFrequest(['a','b','a','c','a']))
~~~
